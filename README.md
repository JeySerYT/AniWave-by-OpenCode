# AniWave 🎬

Аниме каталог с красивым дизайном в тёмной теме.

![AniWave](https://img.shields.io/badge/AniWave-Anime%20Catalog-red?style=for-the-badge)

## ✨ Возможности

- 🔍 Поиск аниме с фильтрами (жанр, год, тип, статус)
- 🏠 Главная страница с трендовыми, популярными и сезонными аниме
- 📺 Hero-секция с трейлерами
- ❤️ Избранное (сохраняется в localStorage)
- 👤 Профиль с кастомизацией (баннер, аватар)
- 🌐 Переключение языка RU/EN
- 📱 Адаптивный дизайн

## 🛠 Технологии

- **React** + **Vite**
- **AniList API** (GraphQL)
- **Apollo Client**
- **Framer Motion** (анимации)
- **React Router** (навигация)
- **CSS** (кастомные стили)

## 🚀 Установка

```bash
# Клонирование репозитория
git clone https://github.com/USER/anwave.git
cd anwave

# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```

## 📁 Структура проекта

```
src/
├── api/           # Apollo Client и GraphQL запросы
├── components/    # UI компоненты
├── context/       # React Context (язык)
├── hooks/         # Кастомные хуки
├── locales/       # Переводы
├── pages/         # Страницы
├── styles/        # Глобальные стили
└── assets/        # Статические файлы
```

## 🎨 Дизайн

- Тёмная тема с красно-розовыми акцентами
- Плавные анимации
- Адаптивная вёрстка

Используется [AniList GraphQL API](https://anilist.co/graphql) — бесплатный, без ключа.
